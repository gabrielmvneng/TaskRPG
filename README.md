# TaskRPG

TaskRPG is a simple command-line task manager with RPG-inspired mechanics.

Users can create tasks, complete them to earn experience points and coins, level up, and buy upgrades from an in-game shop.

## Features

- Add and remove tasks
- Earn XP by completing tasks
- Level up
- Earn coins
- Buy upgrades in the shop

## Built With

- Python 3
- Object-Oriented Programming (OOP)

## Running the project

```bash
python main.py
```

## Why I built this

I created this project to practice object-oriented programming and improve my Python skills by building something fun and interactive.
